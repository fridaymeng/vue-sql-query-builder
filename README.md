# vue-sql-query-builder

## For vue-sql-query-builder.

### How to use

```bash
yarn add vue-sql-query-builder
```
OR
```bash
npm install vue-sql-query-builder
```

### Demo
```bash
yarn start
```

### Init
```js
import * as queryBuilder from "vue-sql-query-builder";
const queryBuilder = new NodeLayout();
queryBuilder.init({
  id: "wrap"
})
```
